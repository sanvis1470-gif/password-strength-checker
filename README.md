A simple application that evaluates password strength in real-time using JavaScript

Features:
- real time password strength detection
- displays strength as weak, medium, or strong
- provides helpful feedback messages
- color-coded strength indicator
- clean and responsive UI

How it works:
the password strength is determined based on the length of the password:
- less than 6 characters - weak
- 6-9 characters - medium
- 10 or more characters - strong
the strength updates automatically as the user types

Tech Stack:
- HTML
- CSS
- JavaScript(DOM manipulation and event handling)

Preview:
- Enter a password in the input field and the app will:
- show strength level
- change text color accordingly
- display helpful suggestions
